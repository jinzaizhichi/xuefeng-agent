## 技术架构

```
用户提问 → AI 提取(省/位次/专业) → 本地数据库查冲稳保 → 联网搜索补充
                                    ↓
                          关键词过滤专业对口
                                    ↓
                           LLM 生成建议 → 浏览器展示
```

- **Web UI**：单文件 HTML+JS，内嵌 Python HTTP 服务器
- **模型无关**：任何 OpenAI 兼容 API（DeepSeek / 通义千问 / GLM / GPT 随意换）
- **离线可用**：数据库在本地，不依赖任何远程服务（联网搜索可选）

## 常见问题

**Q：数据准不准？**
A：数据库来自各省考试院官方投档线，经多轮联网交叉验证。但录取数据每年变，最终决定请查学校官网和考试院。

**Q：怎么更新？**
A：重新下载 ZIP 覆盖文件夹即可。新数据库和新功能自动生效。

**Q：支持哪些 AI？**
A：DeepSeek（推荐）、通义千问、智谱 GLM、GPT-4o 等。API 设置里改 Base URL 和 Model 即可。

**Q：要钱吗？**
A：Agent 本身免费开源。AI 调用按量计费——DeepSeek 新用户送免费额度，聊几百次才几毛钱。
