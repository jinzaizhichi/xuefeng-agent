# 雪峰Agent 使用教程

> 5 分钟，从下载到用上。

---

## 1. 装 Python

按 `Win+R` → 输 `cmd` → 输 `python --version`

出现 3.10 以上 → **跳到第 2 步**。没出现 → 去 **python.org** 下载安装，⚠️ **勾选 "Add Python to PATH"**。

---

## 2. 下载

打开 **github.com/ziqihe10-droid/xuefeng-agent** → 点绿色 **Code** → **Download ZIP** → 解压到桌面。

---

## 3. 搞个 Key

打开 **platform.deepseek.com** → 注册 → 左侧 **API Keys** → 创建 → 复制 `sk-` 开头的那串（只显示一次！）。

---

## 4. 启动

双击 `启动.bat` → 浏览器自动打开 → 点右上角红色 **API设置**：

| 栏位 | 填什么 |
|------|--------|
| Base URL | `https://api.deepseek.com` |
| API Key | `sk-你的钥匙` |
| Model | `deepseek-chat` |
| Tavily Key | 选填（填了搜得更准，去 **tavily.com** 免费注册） |

点 **保存并测试**，绿了就好。

---

## 5. 开问

输入框里跟聊天一样打字：

```
四川物理类619分，位次13420，物化生，普通家庭，
想学自动化机械电气，不接受调剂，冲211，帮我盘盘
```

说清楚：省份、科类、分数、位次、想学什么、不要什么、家里情况。越详细越准。

---

**以后每次用**：双击 `启动.bat`。配置自动记住。

---

## 常见问题

**启动闪退？** → Python 没装好，重装并勾 Add to PATH

**按钮没反应？** → `Ctrl+Shift+R` 强制刷新

**连接失败？** → 检查 Key 前后有没有空格、有没有过期

**数据准吗？** → 24 省考试院官方投档线，42 万条，多轮验证分数 0 误差。最终以官方为准。
