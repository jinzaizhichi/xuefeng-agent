# 雪峰Agent 使用教程 —— 给完全不懂电脑的人

> 不需要懂编程。下载 → 双击 → 浏览器打开 → 直接问。5分钟搞定。

---

## 第一步：安装 Python（已有可跳过）

按 `Win + R`，输入 `cmd`，回车。输入：

```
python --version
```

出现 `Python 3.10` 以上 → **跳到第二步**。出现"不是内部命令" → 往下看。

1. 打开 **python.org**，点黄色 **Download Python**
2. 下载完双击，⚠️ **勾选底部的 "Add Python to PATH"**（必须勾！）
3. 点 **Install Now**，等 1-2 分钟

---

## 第二步：下载雪峰Agent

1. 打开 https://github.com/ziqihe10-droid/xuefeng-agent
2. 点绿色 **Code** → **Download ZIP**
3. 解压到桌面，文件夹名叫 `xuefeng-agent`

---

## 第三步：获取 API Key

推荐 DeepSeek，新用户有免费额度。

1. 打开 https://platform.deepseek.com ，注册登录
2. 左侧点 **API Keys** → **创建 API Key**
3. 复制那串 `sk-` 开头的钥匙（只显示一次！）

---

## 第四步：双击启动

1. 双击文件夹里的 **启动.bat**
2. 会自动弹出浏览器，打开 `http://127.0.0.1:8765/`
3. 点右上角红色 **API设置** 按钮
4. 填入：
   - Base URL: `https://api.deepseek.com`
   - API Key: `sk-你的钥匙`
   - Model: `deepseek-chat`
5. 点 **保存并测试**，显示"连接OK"即可

> 以后每次用，双击 启动.bat 就行。不用重复设置。

---

## 怎么问？

跟聊天一样，直接把你的情况打进去：

```
四川物理类619分，位次13420，物化生，普通家庭，
想学自动化、机械、电气，不接受调剂，冲211，帮我盘盘
```

它会：
1. 自动提取你的省份、位次、专业偏好
2. 查本地数据库（覆盖24省官方录取数据）
3. 联网搜索补充最新信息
4. 按省份志愿政策给你冲稳保建议

---

## 两个模式

| 模式 | 用途 |
|------|------|
| **报考** | 高考志愿填报，查录取数据，盘冲稳保 |
| **娱乐** | 张雪峰风格聊天，减压吐槽 |

顶部按钮切换。默认是报考模式。

---

## 特色功能

- **数据库查询**：内置 24 省 × 2024-2025 年官方投档线（42 万条），自动匹配你的位次
- **联网搜索**：API 设置里填 Tavily Key（可选）可以 AI 联网搜索最新信息；不填则用内置百度搜索
- **专业过滤**：说"想学计算机"，只给你推计算机相关学校，不会混进中医学、法学
- **省份政策感知**：自动识别你的省份，浙江给 80 个志愿建议，老高考给 8-12 所

## 常见问题

**Q：双击启动.bat 没反应？**
A：没装 Python。回去看第一步。

**Q：按钮点了没反应？**
A：强制刷新浏览器（Ctrl+Shift+R）。还不行就关掉浏览器重开。

**Q：提示"请先设置 API Key"？**
A：点右上角红色 API设置按钮，填入 Key。

**Q：回答的专业不对口？**
A：把想学的专业说清楚，比如"只想学计算机、软件、电子，别的不要"。系统会自动过滤。

**Q：数据准确吗？**
A：本地数据库来自各省考试院官方投档线，多位次交叉验证通过。联网搜索数据标注"仅供参考"。最终决定请查学校官网。

**Q：关了浏览器就没了？**
A：不会。数据存在浏览器本地，下次打开还在。点左侧"新建对话"可以开新话题。

## 进阶：Tavily 联网搜索（可选）

Tavily 是专为 AI 设计的搜索引擎，比百度更精准。

1. 打开 https://tavily.com ，注册获取 API Key（有免费额度）
2. 在 API 设置里填入 Tavily Key
3. 以后每次提问，会自动用 Tavily 搜索最新分数线、就业数据

不填也没关系，系统会用内置百度搜索兜底。
