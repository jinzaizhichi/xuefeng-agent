# 雪峰Agent 使用教程 —— 从下载到填报，手把手教会你

> 不需要懂电脑。跟着做就行。全程只需要：**下载、双击、粘贴、打字**。

---

## 快速上手（1 分钟版）

1. 装 Python → 2. 下载解压 → 3. 申请 DeepSeek Key → 4. 双击 `启动.bat` → 5. 填 Key → 6. 打字提问

下面每一步都有详细说明，哪里不会看哪里。

## 第一步：安装 Python

**Python 是什么？** 可以理解成"让这个程序能跑起来的环境"，装一次就行。

### 1. 检查装过没有

键盘按 `Win + R`（Win 键就是左下角那个 Windows 图标键），弹出小窗口，输入 `cmd`，回车。

会弹出一个黑色窗口，在里面输入：

```
python --version
```

如果出现 `Python 3.10` 或 `3.11` 或 `3.12` 或 `3.13` → **你已经装过了，跳到第二步！**

如果出现"不是内部命令"或"不是可运行的程序" → 往下看，装一下。

### 2. 安装步骤

1. 打开浏览器（Edge/Chrome 都行），地址栏输入：**`python.org`**
2. 点页面中间黄色大按钮 **Download Python**（会自动识别你的 Windows）
3. 下载完是一个 `.exe` 文件，双击打开
4. ⚠️ **关键一步**：安装界面底部有个白色小方框 **"Add Python to PATH"**，**一定要勾上！** 不然后面用不了
5. 点 **Install Now**，等蓝色进度条跑完（1-2 分钟）
6. 再按 `Win+R` → `cmd` → `python --version`，显示版本号就对了

---

## 第二步：下载雪峰Agent

1. 打开浏览器，输入：**`https://github.com/ziqihe10-droid/xuefeng-agent`**
2. 找到页面右侧绿色的 **Code** 按钮，点一下
3. 弹出小菜单，点最下面的 **Download ZIP**
4. 浏览器底部会显示下载进度，下载完打开下载文件夹
5. 找到 `xuefeng-agent-main.zip`，**右键 → 全部解压缩 → 解压到桌面**
6. 桌面上会出现一个叫 `xuefeng-agent-main` 的文件夹，打开它

> 里面有一堆文件，不用管，找到 `启动.bat` 就行。

---

## 第三步：获取 API Key

**API Key 是什么？** 就是 AI 大模型的"钥匙"。Agent 本身不包含 AI，需要通过 Key 连接云端 AI。

推荐 **DeepSeek**，便宜好用，注册就送免费额度，够用很久。

1. 浏览器打开：**`https://platform.deepseek.com`**
2. 右上角点 **注册**（手机号就行）
3. 注册完登录，界面左侧找到 **API Keys** 菜单
4. 点 **创建 API Key**，随便起个名（比如"雪峰"）
5. ⚠️ 弹出一串 `sk-` 开头的字符，**立刻复制保存！** 关掉后就再也看不到了
6. 贴到记事本里先存着，下一步要用

---

## 第四步：启动 + 设置

### 启动

1. 打开解压出来的 `xuefeng-agent-main` 文件夹
2. 找到 **`启动.bat`**，双击它
3. 会弹出一个黑色窗口（不要关），等 2-3 秒
4. 浏览器会自动打开 `http://127.0.0.1:8765/`

> 如果浏览器没自动打开，手动打开浏览器输入 `http://127.0.0.1:8765/` 就行。

### 设置 API Key

浏览器页面打开后：

1. 右上角有个红色按钮 **API设置**，点它
2. 弹出一个设置框，有三行要填：

| 字段 | 填什么 |
|------|--------|
| Base URL | `https://api.deepseek.com` |
| API Key | `sk-你的钥匙`（第三步复制的那个） |
| Model | `deepseek-chat` |

3. 还有一行 **Tavily Key**，是联网搜索用的，可以先不填（后面会讲）
4. 点 **保存并测试**
5. 看到绿色 **"连接OK"** → 搞定！

> ⚠️ Key 粘贴时注意前后别有空格。如果显示红色报错，检查 Key 有没有复制完整。

### 以后每次用

双击 `启动.bat` 就行。API 设置浏览器会记住，不用重复填。

## 第五步：开始提问

页面中间有个输入框，跟微信聊天一样，直接打你的情况：

### 示例 1：完整情况

```
四川物理类619分，位次13420，选科物化生。农村普通家庭，
父母工薪，无法支撑长期读研。动手能力强，喜欢拆机械。
想学自动化、机械、电气，其次航空航天、计算机。
完全不接受生化环材。优先武汉、北京、哈尔滨。
不接受调剂，必须一志愿录目标工科。冲211，看重就业高薪。
帮我盘盘冲稳保。
```

### 示例 2：简单问

```
河南文科520分，位次3万多，家里普通，想学个好就业的专业
```

### 示例 3：指定学校

```
山东物理类630分，想冲哈工大自动化，帮我看看有没有希望
```

Agent 会：
1. 自动提取你的省份、位次、专业偏好
2. 查本地数据库（24 省 × 2024-2025 年官方投档线，42 万条）
3. 联网搜索补充最新信息
4. 按你所在省的志愿政策给出冲稳保建议

## 页面功能说明

### 左侧：对话列表
- 每个对话独立保存，不会混在一起
- 点 **+ 新建对话** 可以开新话题
- 点 **x** 可以删除某个对话

### 顶部按钮
- **报考 / 娱乐**：切换模式。报考是正经志愿分析，娱乐是张雪峰风格聊天
- **🌓**：切换深色/浅色模式
- **API设置**：修改 Key 或换 AI 模型

### 底部输入框
- 打字 → 点 **发送** 或按回车
- Shift+回车可以换行

## 提问技巧

### ✅ 这样问最好：

说清楚这几点，AI 能给更精准的建议：

1. **省份 + 科类**：四川物理类 / 河南文科 / 浙江综合
2. **分数 + 位次**：两个都说最好，位次比分数更重要
3. **选科**：物化生 / 史政地
4. **想学什么**：自动化、计算机、电气...
5. **不学什么**：不接受生化环材、不接受师范...
6. **地域偏好**：想去武汉、北京 / 不想去偏远地区
7. **家庭情况**：普通工薪 / 能支持读研 / 有人脉考公
8. **硬性要求**：不接受调剂 / 必须冲 211 / 只考虑公办

### ❌ 这样问效果差：

```
推荐几个学校
```

什么都没说，AI 没法给你针对性的建议。

## 常见问题

**Q：点启动.bat，黑窗闪一下就没了？**
A：Python 没装或者没勾 "Add to PATH"。重装 Python 并确保勾上那个选项。

**Q：浏览器打开了但是页面空白？**
A：等几秒，服务器正在启动。刷新一下。

**Q：按钮点了没反应？**
A：按 `Ctrl+Shift+R` 强制刷新。还是一样就关掉浏览器重开。

**Q：显示"请先设置 API Key"？**
A：点右上角红色 API设置 按钮，填 Key。Key 是 `sk-` 开头的完整一串。

**Q：API 测试显示"连接失败"？**
A：检查三点：
1. Key 前后有没有多余空格
2. Key 有没有过期或没钱（去 DeepSeek 平台查）
3. Base URL 是不是 `https://api.deepseek.com`

**Q：推荐的专业不太对？**
A：在问题里说清楚"只要 XX 专业，别的不要"。AI 会自动过滤。

**Q：数据准不准？**
A：本地数据库来自各省考试院 2024-2025 年官方投档线，经过多轮联网交叉验证，分数 0 误差。但录取数据每年变，最终决定请查学校官网和考试院。

**Q：数据库覆盖哪些省？**
A：浙江、河北、山东、重庆、黑龙江、江苏、湖南、湖北、北京、天津、上海、广东、河南、福建、陕西、甘肃、青海、宁夏、江西、内蒙古、海南、山西、云南、安徽 —— 共 24 省。

**Q：关了浏览器对话会丢吗？**
A：不会。对话保存在浏览器本地，下次打开还在。除非你手动清除浏览器缓存。

**Q：能用手机吗？**
A：目前只支持电脑（Windows）。Mac 也能用但需要终端操作。

**Q：怎么更新到最新版？**
A：去 GitHub 重新下载 ZIP 覆盖文件夹。新数据库和新功能会自动生效。

**Q：想换别的 AI（比如通义千问）？**
A：API 设置里改 Base URL 和 Model 就行。千问的 Key 去阿里云申请。

## 进阶：Tavily 联网搜索（可选但推荐）

Tavily 是专为 AI 设计的搜索引擎，搜出来的结果比百度干净、准确。

1. 打开 **`https://tavily.com`**，点 **Get Started** 注册
2. 进控制台，复制 API Key（`tvly-` 开头）
3. 点 Agent 右上角 API设置 → 填入 Tavily Key 那栏
4. 保存。以后提问时会自动联网搜索最新信息和多来源交叉验证

不填也能用，系统会用内置百度搜索兜底。但填了效果更好。

## 安全提醒

- 你的 API Key 存在浏览器本地，不会上传到任何地方
- 所有数据（对话记录、设置）都在你自己的电脑上
- 没有任何远程服务器收集你的信息
- 如果不放心，去 DeepSeek 平台设置 Key 的消费限额
